# Scheme Navigator for Bharat

**Team:** Deadlock Club  
**Hackathon:** AI for Bharat (AWS)

AI-assisted eligibility checker & scheme recommender that helps users:
- Check eligibility using explainable JSON rules
- Get ranked scheme recommendations
- Understand why they are/aren’t eligible
- Access official links and required documents

## Files
- `requirements.md` — Kiro-generated requirements
- `design.md` — Kiro-generated design

## Flow
Profile → Recommend → Eligibility Check → AI Explanation → Official Links
